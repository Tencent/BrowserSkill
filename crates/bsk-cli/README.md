# bsk

Command-line interface and background daemon for [BrowserSkill](https://github.com/Tencent/BrowserSkill).

Install:

```bash
curl -fsSL https://raw.githubusercontent.com/Tencent/BrowserSkill/main/install.sh | sh
```
